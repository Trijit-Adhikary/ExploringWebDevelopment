# Cursor Landing Page Clone

A responsive clone of the Cursor IDE landing page built with HTML and CSS. This project demonstrates modern web design practices with a dark theme, smooth interactions, and organized component-based styling.

## Features

- **Sticky Navigation Header** - Always accessible navigation with sign-in and download CTAs
- **Hero Section** - Eye-catching headline with product screenshot and call-to-action
- **Company Logos** - Grid of trusted company logos
- **Features Section** - Three showcase sections with alternating image-text layouts demonstrating key capabilities
- **Testimonials** - Six testimonial cards showcasing user feedback
- **Stories Section** - Three featured story cards with images and descriptions
- **Changelog** - Recent version updates and release notes
- **Team Section** - Team introduction with photo
- **Highlights** - Featured blog posts and announcements
- **Final CTA** - Download call-to-action section
- **Footer** - Comprehensive link navigation organized by category

## Design Details

- **Color Scheme**: Dark theme with orange accent color (`#f54e00`)
- **Typography**: CursorGothic font with system font fallbacks
- **Layout**: Centered max-width container (1300px) with responsive padding
- **Interactions**: Hover effects on buttons, cards, and interactive elements
- **Responsiveness**: Mobile-friendly design with proper spacing

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone or download this repository
2. Navigate to the `Cursor_landing_page_clone` directory
3. Open `Cursor_Clone.html` in your web browser

### Files Overview

| File | Purpose |
|------|---------|
| `Cursor_Clone.html` | Main HTML structure with semantic markup |
| `style.css` | Complete styling with CSS variables and organized sections |
| `media/` | Directory containing all image and SVG assets |

## Usage

Simply open the HTML file in any web browser. All styling is contained in the CSS file, and all assets are loaded from the relative `media/` directory.
